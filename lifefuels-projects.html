LifeFuels Internship

About

An on-the-go keurig like water bottle that dispenses vitamins and nutrients based on your needs and keeps track of it: that is what LifeFuel's main product is. LifeFuel's goal is to help people easily hydrate and get the proper nutrition. Check out the company by clicking here.

I had the opportunity to intern at this wonderful company for two summers. I coded some webpages for the first internship and worked on an RFID scanner for the second.

Lentil

About

Lentil is an RFID scanner that uses an Arduino Uno and a computer to write data to an RFID tag. When I got to tackle this project, the machine only wrote hard-coded data into the tag. My manager tasked me to code the machine to make it read from a file that contains data and write it into the RFID tag.

End Product

The Arduino was sucessfully able to receive messages from the computer to write the data into the tag. It took six seconds for Lentil to write twenty-seven pages of data to the tag. Check out the notes I wrote below!

Website

About

I wasn't sure if I wanted to go into computer science or computer engineering. Given different options between front-end, back-end, and firmware, I chose front-end because that seemed to be the easiest. Prior to this internship, I had no website coding knowledge; I had to learn the language and syntax through external sources.

End Product

The three pages I coded were the team page, about page, and part of the product page. I don't have an example of my work because I didn't document it (sorry).

In the team page, each person had a card. Each card had a picture, a gif, their name, and icons that would link to their social websites. When the user hovers over the person's picture, a gif would play. The about page had a banner at the top and a bunch of text after it; this was a simple page. The product page was the hardest to implement. It read from a file that contained information about the product and displayed it on the website.
